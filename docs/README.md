### Quick Links

```bash
# Install
git clone https://github.com/Gokul287/Review-Pilot.git && cd Review-Pilot && npm install

# Analyze your changes
npx reviewpilot check

# Auto-fix issues
npx reviewpilot fix --dry-run

# Create PR
npx reviewpilot create-pr
```
